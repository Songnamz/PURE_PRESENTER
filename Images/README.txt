IMAGES FEATURE
==============

This folder stores all imported images for display during church services.

HOW TO USE:
-----------

1. IMPORT IMAGES:
   - Open the "Church Program" window (Project List)
   - Expand the "Images" folder
   - Click "Import Images" button
   - Select one or more image files
   - Files will be copied to this Images folder

2. DISPLAY IMAGES:
   - Click on any image name in the list
   - Image will display fullscreen on Display2 (projector window)
   - Perfect for announcements, photos, graphics, etc.

3. DELETE IMAGES:
   - Click the trash icon (üóëÔ∏è) next to any image
   - Confirm the deletion
   - Image will be removed from the folder

4. SEARCH:
   - Use the search box to filter images by name

SUPPORTED FORMATS:
------------------
- JPEG (.jpg, .jpeg)
- PNG (.png)
- GIF (.gif)
- BMP (.bmp)
- WebP (.webp)
- SVG (.svg)
- TIFF (.tiff, .tif)
- ICO (.ico)
- HEIC (.heic)
- HEIF (.heif)

COMMON USES:
------------
- Church announcements
- Event posters
- Worship backgrounds
- Bible verse graphics
- Photos of church activities
- Sermon illustrations
- Prayer requests
- Welcome slides
- Offering/giving information
- Contact information

NOTES:
------
- Images are stored locally in this folder
- Multiple files can be imported at once
- Images display fullscreen on Display2
- Click any image name to display it immediately
- High resolution images work best for projection
- Images maintain aspect ratio (no cropping)

TIPS:
-----
- Use 1920x1080 (Full HD) resolution for best results
- PNG format supports transparency
- Name files descriptively for easy searching
- Test image display before the service
- Keep frequently used images at the top by naming them with numbers
